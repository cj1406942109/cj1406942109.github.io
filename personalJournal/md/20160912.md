**jQuery封装与zepto封装**
jQuery的插件封装模板：
```js
(function($) {
$.fn.extend({...});
}(jQuery); //参数可以写jQuery，也可以写 window.jQuery
```

兼容zepto和jQuery的插件封装模板：
```js
(function($) {
$.extend($.fn ,{...});
}(window.jQuery || window.Zepto); //参数不能省略window.
```
