### 学习内容

#### **git版本回滚**
1. `git log` 
2. `git reset --hard 版本号`
3. `git push -f`
```git
git log         //查看历史版本信息
git reset --hard 版本号       //回退到版本号对应的版本
git push -f         //强推
```
