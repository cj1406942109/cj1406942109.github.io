# nvm使用说明
> 记录时间：2016-11-11

```bash
  # 查看可以下载使用的node版本
  nvm ls-remote

  # 安装某个版本的node
  nvm install v4.6.2		# 安装版本为4.6.2的node.js

  # 设置默认版本的nodejs
  nvm alias default v4.6.2		# 将版本为4.6.2的node.js设置为默认版

  # 查看已安装的node版本
  nvm ls

  # 切换当前使用的node版本
  nvm use v4.6.2

  # 查看当前使用的node版本
  nvm current
```
